# Backstage Design System

The Backstage Design System is a collection of reusable UI components built for Codelitt's internal tools. It provides a consistent, accessible, and well-documented component library.

## Getting Started

Browse the component documentation in the sidebar to explore all available components. Each component page includes:

- **Interactive controls** — adjust props in real time
- **Code examples** — copy-paste ready snippets
- **Variants & states** — see every visual permutation

## Component Categories

### Components
Core building blocks like **Button**, **Card**, **SectionHeader**, and **Tag**.

### Form
Form controls including **FormInput**, **FormLabel**, **FormSelect**, and **SearchableSelect**.

### Feedback
User feedback components: **Toast**, **ErrorAlert**, **LoadingSpinner**, **EmptyState**, and **Tooltip**.

### Data Display
Data presentation components: **StatusIndicator** and **Pagination**.

## Design Principles

- **Consistency** — Shared color palette, spacing, and border radius across all components
- **Accessibility** — Semantic HTML, ARIA attributes, focus styles, and keyboard navigation
- **Simplicity** — Minimal API surface with sensible defaults
- **Composability** — Components work together naturally

## Tech Stack

- **React 19** with TypeScript
- **Tailwind CSS 4** for styling
- **Storybook 8** for documentation and development
